1.еҜјиҜӯ

вҖӢ йҰ–е…ҲжқҘз®ҖеҚ•д»Ӣз»ҚдёҖдёӢwebpackпјҡзҺ°д»Ј JavaScript еә”з”ЁзЁӢеәҸзҡ„йқҷжҖҒжЁЎеқ—жү“еҢ…е·Ҙе…·гҖӮеҪ“ webpack еӨ„зҗҶеә”з”ЁзЁӢеәҸж—¶пјҢе®ғдјҡеңЁеҶ…йғЁжһ„е»әдёҖдёӘдјҡжҳ е°„йЎ№зӣ®жүҖйңҖзҡ„жҜҸдёӘжЁЎеқ— зҡ„дҫқиө–еӣҫ(dependency graph)пјҢе№¶з”ҹжҲҗдёҖдёӘжҲ–еӨҡдёӘ bundleгҖӮwebpack4.0еҮәзҺ°д№ӢеҗҺпјҢжҲ‘д»¬еҸҜд»ҘдёҚз”ЁеҶҚеј•е…ҘдёҖдёӘй…ҚзҪ®ж–Үд»¶жқҘжү“еҢ…йЎ№зӣ®пјҢе№¶дё”е®ғд»Қз„¶жңүзқҖеҫҲй«ҳзҡ„еҸҜй…ҚзҪ®жҖ§пјҢеҸҜд»ҘеҫҲеҘҪж»Ўи¶іжҲ‘д»¬зҡ„йңҖжұӮгҖӮ
еңЁејҖе§ӢжӯЈж–Үд№ӢеүҚпјҢйҰ–е…Ҳе…ҲжқҘзңӢзңӢжҲ‘д»¬иҰҒе®һзҺ°зҡ„жҲҗжһңпјҡ

  • ж”ҜжҢҒES6+JQuery+Less/Scssзҡ„еҚ•йЎө/еӨҡйЎөи„ҡжүӢжһ¶
  • ж”ҜжҢҒES6+React+Less/Scss+Typescriptзҡ„еҚ•йЎө/еӨҡйЎөи„ҡжүӢжһ¶
  • ж”ҜжҢҒES6+Vue+Less/Scss+Typescriptзҡ„еҚ•йЎө/еӨҡйЎөи„ҡжүӢжһ¶

githubең°еқҖпјҡ

еҹәдәҺwebpack4.0жҗӯе»әзҡ„и„ҡжүӢжһ¶пјҲж”ҜжҢҒreact/vue/typescript/es6+/jquery+less/scss)

еңЁи„ҡжүӢжһ¶зҡ„ејҖеҸ‘иҝҮзЁӢдёӯжҲ‘дјҡиҜҰз»Ҷд»Ӣз»ҚжҜҸдёӘжҸ’д»¶жҲ–иҖ…loaderзҡ„з”ЁйҖ”д»ҘеҸҠwebpackзҡ„ж ёеҝғзҗҶеҝөпјҢеҰӮжңүдёҚжҮӮжҲ–иҖ…жңүе…¶д»–жӣҙеҘҪзҡ„жғіжі•ж¬ўиҝҺдәӨжөҒгҖӮ
дёӢйқўжҳҜеҹәдәҺиҜҘж–Үз« зҡ„webpack4.0зҡ„жҖқз»ҙеҜјеӣҫпјҡ

2.webpackж ёеҝғжҰӮеҝө

  • е…ҘеҸЈпјҡжҢҮзӨә webpack еә”иҜҘдҪҝз”Ёе“ӘдёӘжЁЎеқ—дҪңдёәе…ҘеҸЈиө·зӮ№

  • иҫ“еҮәпјҡе‘ҠиҜү webpack еңЁе“ӘйҮҢиҫ“еҮәе®ғжүҖеҲӣе»әзҡ„ bundleпјҢд»ҘеҸҠеҰӮдҪ•е‘ҪеҗҚиҝҷдәӣж–Үд»¶

  • loaderпјҡи®© webpack иғҪеӨҹеҺ»еӨ„зҗҶе…¶д»–зұ»еһӢзҡ„ж–Үд»¶пјҢе№¶е°Ҷе®ғд»¬иҪ¬жҚўдёәжңүж•ҲжЁЎеқ—пјҢд»Ҙдҫӣеә”з”ЁзЁӢеәҸдҪҝз”Ё

  • жҸ’д»¶пјҡз”ЁдәҺжү§иЎҢиҢғеӣҙжӣҙе№ҝзҡ„д»»еҠЎгҖӮеҢ…жӢ¬пјҡжү“еҢ…дјҳеҢ–пјҢиө„жәҗз®ЎзҗҶпјҢжіЁе…ҘзҺҜеўғеҸҳйҮҸ

  • жЁЎејҸпјҡйҖҡиҝҮйҖүжӢ© development, production жҲ– none д№Ӣдёӯзҡ„дёҖдёӘпјҢжқҘи®ҫзҪ® mode еҸӮж•°пјҢд»ҺиҖҢиҝӣиЎҢдёҚеҗҢзҡ„жү“еҢ…дјҳеҢ–

  • жөҸи§ҲеҷЁе…је®№жҖ§пјҡж”ҜжҢҒжүҖжңүз¬ҰеҗҲES5 ж ҮеҮҶзҡ„жөҸи§ҲеҷЁпјҲдёҚж”ҜжҢҒ IE8 еҸҠд»ҘдёӢзүҲжң¬пјү
    дёӢйқўжҸҗдҫӣе®ҳзҪ‘зҡ„жү“еҢ…жЁЎеһӢ

3.ж”ҜжҢҒES6+JQuery+Less/Scssзҡ„еҚ•йЎө/еӨҡйЎөи„ҡжүӢжһ¶

еңЁе®һзҺ°и„ҡжүӢжһ¶д№ӢеүҚпјҢеҒҮи®ҫжҲ‘д»¬е·Із»ҸеҲӣе»әдәҶзӣ®еҪ•е’Ңpackage.jsonж–Үд»¶пјҢжҺҘдёӢжқҘе…Ҳе®үиЈ…webpackзӣёе…ідҫқиө–пјҡ

еӣ дёәйЎ№зӣ®иҰҒж”ҜжҢҒes6+,жҲ‘д»¬иҝҳйңҖиҰҒе®үиЈ…babelзӣёе…ідҫқиө–пјҡ

иҝҷдёӘж—¶еҖҷеҸҜд»ҘејҖе§Ӣй…ҚзҪ®жҲ‘д»¬зҡ„и„ҡжүӢжһ¶йҖ»иҫ‘дәҶпјҢдёәдәҶйЎ№зӣ®з»“жһ„жё…жҷ°жҳ“дәҺз»ҙжҠӨпјҢжҲ‘д»¬е»әдёҖдёӘbuildзӣ®еҪ•дё“й—Ёж”ҫwebpackжһ„е»әзҡ„и„ҡжң¬пјҢwebpackй»ҳи®Өзҡ„й…ҚзҪ®ж–Үд»¶жҳҜwebpack.config.js,з”ұдәҺе®һйҷ…йЎ№зӣ®йңҖиҰҒпјҢжҲ‘д»¬е°Ҷе…¶жӢҶеҲҶдёә3дёӘж–Үд»¶пјҢеҲҶеҲ«жҳҜwebpackйҖҡз”Ёй…ҚзҪ®ж–Үд»¶webpack.base.jsпјҢејҖеҸ‘зҺҜеўғй…ҚзҪ®ж–Үд»¶webpack.dev.jsд»ҘеҸҠз”ҹдә§зҺҜеўғй…ҚзҪ®ж–Үд»¶webpack.prod.jsгҖӮ

жҲ‘д»¬е…ҲеӨ„зҗҶwebpack.common.jsж–Үд»¶

дёәдәҶйЎ№зӣ®еҗҺжңҹзҡ„ејҖеҸ‘е’Ңз»ҙжҠӨпјҢжҲ‘д»¬е»әз«ӢеҘҪйЎ№зӣ®з»“жһ„пјҡ

publicзӣ®еҪ•жҳҜдәӢе…ҲеҮҶеӨҮеҘҪзҡ„htmlжЁЎзүҲпјҢиҝҷйҮҢе°ұдёҚд»Ӣз»ҚдәҶпјҢе…¶д»–зӣ®еҪ•еҸҜж №жҚ®е…·дҪ“йЎ№зӣ®иҝӣиЎҢи®ҫзҪ®гҖӮ

жҲ‘д»¬иҝҳйңҖиҰҒдёҖдёӘжҸ’д»¶е°Ҷжү“еҢ…еҗҺзҡ„ж–Үд»¶жӨҚе…ҘеҲ°htmlжЁЎзүҲдёӯе№¶еҜјеҮәеҲ°distзӣ®еҪ•дёӢпјҢиҝҷйҮҢдҪҝз”Ёhtml-webpack-pluginжқҘе®һзҺ°

зҺ°еңЁwebpack.base.jsдёәеҰӮдёӢпјҡ

дёәдәҶжү“еҢ…йЎ№зӣ®пјҢжҲ‘д»¬йңҖиҰҒеңЁwebpack.prod.jsзӣ®еҪ•дёӢиҝӣиЎҢй…ҚзҪ®пјҢжӯӨеӨ„йңҖиҰҒдёҖдёӘжЁЎеқ—webpack-mergeе°ҶwepackеҹәзЎҖй…ҚзҪ®еҗҲе№¶иҝӣз”ҹдә§й…ҚзҪ®пјҢжҲ‘д»¬е…ҲжқҘе®үиЈ…дёҖдёӢпјҡ

webpack.prod.jsй…ҚзҪ®еҰӮдёӢпјҡ

з„¶еҗҺеңЁpackage.jsonйҮҢж·»еҠ жү§иЎҢи„ҡжң¬пјҡ

webpackй»ҳи®ӨдјҡжүҫеҗҚдёәwebpack.config.jsзҡ„ж–Үд»¶пјҢз”ұдәҺжҲ‘д»¬е°Ҷе…¶жӢҶи§Јдёәprodе’ҢdevпјҢжүҖд»ҘжҲ‘д»¬иҰҒжүӢеҠЁжҢҮе®ҡwebpackжү§иЎҢзҡ„ж–Үд»¶пјҢж·»еҠ вҖ“configпјҢеҚіеҸҜжүӢеҠЁжҢҮе®ҡзӣ®еҪ•гҖӮдёӢйқўжҲ‘д»¬ејҖе§ӢеҶҷдёҖж®өд»Јз ҒиҜ•иҜ•еҗ§пјҢеңЁindex.jsдёӯеҶҷе…ҘеҰӮдёӢes6д»Јз Ғ:

дёӢйқўжҲ‘д»¬жү§иЎҢпјҡ

жӯӨж—¶жҲ‘д»¬дјҡзңӢи§ҒйЎ№зӣ®дёӯеӨҡдәҶдёҖдёӘdistзӣ®еҪ•пјҢйҮҢйқўзҡ„htmlд№ҹжӨҚе…ҘдәҶзӣёеә”зҡ„д»Јз ҒпјҢ

еңЁжөҸи§ҲеҷЁдёӯжү“ејҖпјҡ

image.png

ok,з¬¬дёҖжӯҘе®ҢжҲҗгҖӮдёӢдёҖжӯҘжҳҜж”ҜжҢҒcssпјҢжҲ‘д»¬е…Ҳе®үиЈ…еҰӮдёӢеҮ дёӘжЁЎеқ—пјҡ

еңЁwebpack.base.jsдёӯзҡ„moduleдёӯж·»еҠ еҰӮдёӢд»Јз Ғпјҡ

жіЁж„ҸпјҢlaoderзҡ„еҠ иҪҪйЎәеәҸжҳҜд»ҺдёӢеҫҖдёҠпјҢд»ҺеҸіеҫҖе·Ұзҡ„пјҢжүҖд»Ҙй…ҚзҪ®loaderзҡ„ж—¶еҖҷиҰҒжіЁж„ҸдёҖдёӢйЎәеәҸгҖӮ
жӯӨж—¶еңЁstylesзӣ®еҪ•дёӢеҠ е…Ҙapp.css,еңЁjsдёӯеј•е…Ҙ:

жӯӨж—¶жү“ејҖжөҸи§ҲеҷЁпјҢеҸҜд»ҘзңӢеҲ°cssз”ҹж•ҲдәҶпјҡ

image.png

зҺ°еңЁcssеҜје…ҘиҷҪз„¶з”ҹж•ҲдәҶпјҢдҪҶжҳҜжҳҜжңүjsеҠЁжҖҒеҲӣе»әж·»еҠ еҲ°headйҮҢйқўзҡ„пјҢеҰӮжһңеҗҺжңҹйЎ№зӣ®еӨҚжқӮдәҶпјҢе°ҶдјҡдёҘйҮҚеҪұе“ҚйЎ№зӣ®зҡ„еҠ иҪҪйҖҹеәҰпјҢжүҖд»ҘжҲ‘д»¬иҝҷйҮҢйңҖиҰҒеҸҰдёҖдёӘжҸ’д»¶пјҢеҜ№cssиҝӣиЎҢд»Јз ҒеҲҶеүІпјҢеҚ•зӢ¬з”ҹжҲҗcssж–Үд»¶пјҡ

ж №жҚ®иҜҘжҸ’д»¶зҡ„е®ҳж–№й…ҚзҪ®пјҢжҲ‘д»¬йңҖиҰҒжҠҠstyle-loaderжӣҝжҚўжҲҗиҜҘжҸ’д»¶жҸҗдҫӣзҡ„loaderпјҢе№¶й…ҚзҪ®еҜјеҮәзҡ„cssж–Үд»¶зӣ®еҪ•е’Ңж–Үд»¶еҗҚпјҢдёәдәҶжҸҗй«ҳејҖеҸ‘зҺҜеўғжһ„е»әйҖҹеәҰпјҢжҲ‘д»¬еҸӘеңЁз”ҹдә§зҺҜеўғеҲҶеүІcssпјҡ

з”ұдәҺжҲ‘д»¬еңЁdevе’ҢprodзҺҜеўғзҡ„css-loaderдёҚдёҖж ·пјҢжүҖд»ҘжҲ‘д»¬е°Ҷbaseзҡ„css-loaderй…ҚзҪ®еҲ йҷӨпјҢз§»еҲ°devдёӢ

ok,жӯӨж—¶жҲ‘д»¬е°ұе®ҢжҲҗдёҖдёӘеҹәжң¬зҡ„жү“еҢ…es6,cssзҡ„жЁЎеқ—жү“еҢ…е·Ҙе…·пјҢдёәдәҶж”ҜжҢҒжӣҙй«ҳзҡ„es6+иҜӯжі•пјҢжҲ‘д»¬й…ҚзҪ®.babelrcж–Үд»¶пјҢд»ҘеҸҠе®үиЈ…зӣёеә”зҡ„npmеҢ…пјҡ

.babelrcж–Үд»¶еҰӮдёӢпјҡ

жҲ‘д»¬дјҡзңӢеҲ°babelrcж–Үд»¶йҮҢйқўжңүвҖқuseBuiltInsвҖқ: вҖңusageвҖқ, иҝҷдёӘй…ҚзҪ®ж¶үеҸҠеҲ°дёҖдёӘе…ідәҺwebpackжү“еҢ…зҡ„й«ҳзә§з”Ёжі•пјҢtree-shakingгҖӮ

tree-shaking:з”ЁдәҺжҸҸиҝ°з§»йҷӨ JavaScript дёҠдёӢж–Үдёӯзҡ„жңӘеј•з”Ёд»Јз Ғ(dead-code)гҖӮе®ғдҫқиө–дәҺ ES2015 жЁЎеқ—иҜӯжі•зҡ„ йқҷжҖҒз»“жһ„ зү№жҖ§пјҢдҫӢеҰӮ import е’Ң exportгҖӮиҝҷдёӘжңҜиҜӯе’ҢжҰӮеҝөе®һйҷ…дёҠжҳҜз”ұ ES2015 жЁЎеқ—жү“еҢ…е·Ҙе…· rollup жҷ®еҸҠиө·жқҘзҡ„гҖӮ
